QDox 1.6.3

QDox is a high speed, small footprint parser for extracting class/interface/method definitions from source files complete with JavaDoc @tags. It is designed to be used by active code generators or documentation tools. For more information, please visit http://qdox.codehaus.org/.

This plugin includes the binary version of QDox 1.6.3, which can be found in the 'lib/' directory of this plugin (qdox-1.6.3.jar as obtained fromhttp://qdox.codehaus.org/download.html).
