Tigerstripe Workbench

Welcome to Tigerstripe Workbench User Manual

This manual is organized as follows:

  • The Install section details the installation process for Tigerstripe Workbench.
  • The Getting Started section covers the basics to get you through your first project with Tigerstripe. It introduces the Tigerstripe Workbench environment within Eclipse and walks through a simple Service Contract definition example.
  • The Concepts section provides a definition of all the concepts introduced by Tigerstripe, including the methodology embedded in the workbench to design an Interface Contract with Tigerstripe.
  • The Tasks section acts as the reference manual for all tasks involved in designing a Service Contract with Tigerstripe. From modeling, documentation to generation, each step of the design is covered.
  • The Troubleshooting section provides hints on how to resolve errors on a Tigerstripe project that may be raised during the design phase.

For details about the specific enhancements and bug fixes from previous versions of Tigerstripe, please read our release notes.
