Named Query Implementation Specifics

The Implementation Specific Details for the artifact can be specified from the Details tab in the Tigerstripe Query Artifact editor. The following information is available in the Details tab:

  • Interface Package: the interface package where the corresponding Java interface is generated. This option only applies to the J2EE Integration Profile.
  • Returned Entity Type: the fully qualified name of the returned entity type.
