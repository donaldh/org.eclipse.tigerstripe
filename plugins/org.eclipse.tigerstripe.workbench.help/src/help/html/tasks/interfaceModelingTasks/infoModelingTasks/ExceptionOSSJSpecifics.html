Exception Artifact OSS/J Specifics

The Implementation Specific Details for the artifact can be specified from the Details tab in the Tigerstripe Exception Artifact editor. The following information is available in the Details tab:

  • Interface Package: the interface package where the corresponding Java interface is generated. This option only applies to the J2EE Integration Profile.
