Enumeration Implementation Specifics

The OSS/J Specific details for the artifact can be specified from the Details tab of the Tigerstripe Enumeration Artifact editor. The following information is available in the Details tab:

  • Interface Package: the interface package where the corresponding Java interface is generated. This option only applies to the J2EE Integration Profile.
  • Base type: the base type for the Enumeration. Valid values include, int and String. These types are mapped accordingly for XML and WSDL generation therefore there is no reference to Java.
  • Is Extensible: this is a hint for code generation to specify whether this enumeration is to be extended or not (that is, if additional constants may be defined or not). The default value is selected.
