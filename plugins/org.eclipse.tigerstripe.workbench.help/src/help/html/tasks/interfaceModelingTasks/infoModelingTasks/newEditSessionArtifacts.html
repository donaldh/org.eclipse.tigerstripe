Editing Session Facade Artifacts

After you successfully create the session facade artifact, the corresponding icon appears in the Tigerstripe Explorer, and the Overview tab displays. You can further edit your newly created session facade artifact in the Overview tab.

Editing a Session Facade Artifact

Note: Make sure that the Tigerstripe Perspective is active.

To edit a session facade artifact:

  1. Double-click the session facade artifact in the Tigerstripe Explorer to open the Tigerstripe Session Facade Artifact editor.
  2. Edit the Session Facade Artifact as necessary and click Save to save your changes. Alternatively, you can select File and click Save or press CTRL+S to save your changes.

You can set the following parameters in the Tigerstripe Session Facade Artifact editor:

  • General Information: Common to all Artifacts, the general information section includes high level information such as version, and description.
  • Methods: You can define methods on your artifact.

Session Facade artifacts also include a set of options to customize the generated code according to OSS/J design guidelines. For more information, refer to Session Facade Implementation Specifics.
