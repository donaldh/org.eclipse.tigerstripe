Editing a Datatype Artifact

After you successfully create the datatype artifact, the datatype icon appears in the Tigerstripe Explorer, and the Overview tab displays. You can further edit your newly created datatype artifact in the Overview tab.

Note: You can create new datatype artifacts and you can also add, remove, or edit datatype artifacts information in the Overview tab.

Editing a Datatype Artifact

Note: Make sure that the Tigerstripe Perspective is active.

To edit a Datatype Artifact:

  1. Double-click the datatype in the Tigerstripe Explorer to open the Tigerstripe Datatype Artifact editor.
  2. Edit the datatype artifact as necessary and click Save to save your changes. Alternatively, you can select File and click Save or press CTRL+S to save your changes.

You can set the following parameters in the Tigerstripe Datatype Artifact editor:

  • General Information: Common to all Artifacts, the general information section covers high level information such as version, description.
  • Attributes: This is where you can define Artifact attributes.
  • Constants: This is where you can define artifact constants.
  • Methods: This is where you can define artifact methods.

Datatype artifacts also include a set of options to customize the generated code according to OSS/J design guidelines. For more information, refer to Datatype Implementation Specifics.
