Editing Entity Artifacts

After you successfully create the entity artifact, the corresponding entity icon appears in the Tigerstripe Explorer, and the Overview tab displays. You can further edit your newly created entity artifact in the Overview tab.

Note: You can create entity attributes and you can also add, remove, or edit attributes on an entity artifact in the Overview tab.

Editing an Entity Artifact

Note: Make sure that the Tigerstripe Perspective is active.

To edit an Entity Artifact

  1. Double-click the entity in the Tigerstripe Explorer to open the Tigerstripe Entity Artifact editor.
  2. Edit the Entity Artifact as necessary and click Save to save your changes. Alternatively, you can select File and click Save or press CTRL+S to save your changes.

You can set the following parameters in the Tigerstripe Entity Artifact editor:

  • General Information: Common to all Artifacts, the general information section includes high level information such as version and description.
  • Entity Attributes: This is where you can define Artifact attributes.
  • Entity Constants: This is where you can define constants for your artifact constants.
  • Entity Methods: This is where you can define artifact methods.

Entity artifacts also include a set of options to customize the generated code according to OSS/J design guidelines. For more information, refer to Entity Implementation Specifics.
