Editing Package Artifacts

Note: Make sure that the Tigerstripe Perspective is active.

To edit a Package Artifact

  1. Right-click the PAckage Artifact in the Tigerstripe Explorer, and select the option "Open Package Artifact In Editor" to open the Tigerstripe Package Artifact editor.
  2. Note: That this is different behaviour form all other artifact types. The Double-click option opens the package contents in the explorer.

You can set the following parameters in the Tigerstripe Package Artifact editor:

  • General Information: Common to all Artifacts, the general information section covers high level information such as version, description.
  • Note: That package artifacts do not support the "Extends" field.
Once you have edited your changes, make sure you Save your changes for them to be taken into account by Tigerstripe. To save, either select the Save option from the File menu entry in the menubar, use the save icon in the toolbar, or hit ctrl-s.
