Session Facade Implementation Specifics

The Session Facade Implementation Specific details for the artifact can be specified from the Details tab in the Tigerstripe Session Facade Artifact editor. The following information is available in the Details tab:

  • Interface Package: the interface package where the corresponding Java interface is generated. This option only applies to the J2EE Integration Profile.
  • Managed Entities: the list of entities marked as managed through the Session Facade artifact.
  • Named Queries: the list of named queries exposed by the Session Facade artifact.
  • Update Procedures: the list of update procedures exposed by the Session Facade artifact.
  • Emitted Notifications: the list of notifications emitted through the Session Facade artifact.
