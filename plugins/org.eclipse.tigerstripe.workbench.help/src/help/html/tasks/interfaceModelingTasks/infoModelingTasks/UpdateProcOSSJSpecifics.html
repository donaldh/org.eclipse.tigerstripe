Update Procedure OSS/J Specifics

The Implementation Specific Details for the artifact can be specified from the Details tab in the Tigerstripe Update Procedure Artifact editor. The following information is available in the Details tab:

  • Interface Package: the interface package where the corresponding Java interface is generated. This option only applies to the J2EE Integration Profile.
    • Single Extension Only: this flag provides a hint for the generation and in particular, acts on the J2EE Integration Profile. The default value is unchecked.
    • Factory methods on Session Facade
