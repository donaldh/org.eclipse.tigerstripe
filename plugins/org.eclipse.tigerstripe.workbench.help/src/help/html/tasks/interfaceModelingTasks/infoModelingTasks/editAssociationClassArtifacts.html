Editing Association Class Artifacts

First, please make sure the Tigerstripe Perspective is active.

To edit an existing Association Class Artifact, simply select its corresponding entry in the Tigerstripe Explorer, and double-click to open it in an editor. Association Class Artifacts are made of the following details:

  • General Information: Common to all Artifacts, the general information section covers high level information such as version, description.
  • Association Ends: In addition to the general items, you can set the end details for the association class.
  • Attributes: This is where you can define Artifact attributes.
  • Methods: This is where you can define Artifact methods.
Once you have edited your changes, make sure you Save your changes for them to be taken into account by Tigerstripe. To save, either select the Save option from the File menu entry in the menubar, use the save icon in the toolbar, or hit ctrl-s.
