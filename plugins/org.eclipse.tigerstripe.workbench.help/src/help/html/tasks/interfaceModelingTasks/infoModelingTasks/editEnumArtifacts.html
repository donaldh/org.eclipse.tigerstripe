Editing an Enumeration Artifact

Note: Make sure that the Tigerstripe Perspective is active.

To edit an Enumeration Artifact:

  1. Double-click the Enumeration Artifact entry in the Tigerstripe Explorer to open the Tigerstripe Enumeration Artifact editor.

You can set the following parameters in the Tigerstripe Enumeration Artifact editor:

  • General Information: Common to all Artifacts, the general information section covers high level information such as version, description.
  • Constants: This is where you can define Artifact literals.

Also, on the Details tab of the editor, you can set the following:

  • Base Type: The base type can take a value of String or int, and all Literals must be of this type. When you add a new Literal it will automaticaly be of this type, but you will need to edit any Literals that exist prior to changing the value here.
Once you have edited your changes, make sure you Save your changes for them to be taken into account by Tigerstripe. To save, either select the Save option from the File menu entry in the menubar, use the save icon in the toolbar, or hit ctrl-s.
