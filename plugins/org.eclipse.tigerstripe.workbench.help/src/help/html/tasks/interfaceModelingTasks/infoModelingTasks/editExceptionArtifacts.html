Editing Exception Artifacts

Note: Make sure that the Tigerstripe Perspective is active.

To edit an Exception Artifact:

  1. Double-click the Exception Artifact in the Tigerstripe Explorer to open the Tigerstripe Exception Artifact editor.

You can set the following parameters in the Tigerstripe Exception Artifact editor:

  • General Information: Common to all Artifacts, the general information section covers high level information such as version, description.
  • Attributes: This is where you can define Artifact attributes.
Once you have edited your changes, make sure you Save your changes for them to be taken into account by Tigerstripe. To save, either select the Save option from the File menu entry in the menubar, use the save icon in the toolbar, or hit ctrl-s.
