After you successfully create the entity artifact, the corresponding entity icon appears in the Tigerstripe Explorer, and the Overview tab displays. You can further edit your newly created enumeration artifact in the Overview tab.

Note: You can create enumeration labels and you can also add, remove, or edit enumeration artifacts on the Overview tab.

Editing an Enumeration Artifact

Note: Make sure that the Tigerstripe Perspective is active.

To edit and Enumeration Artifact:

  1. Double-click the enumeration artifact entry in the Tigerstripe Explorer to open the Tigerstripe Enumeration Artifact editor.
  2. Edit the Enumeration Artifact as necessary and click Save to save your changes. Alternatively, you can select File and click Save or press CTRL+S to save your changes.

You can set the following parameters in the Tigerstripe Enumeration Artifact editor:

  • General Information: common to all Artifacts, the general information section covers high level information such as version, description.
  • Constants: This is where you define Artifact attributes.

Enumeration artifacts also include a set of options to customize the generated code according to OSS/J design guidelines. For more information, refer to Enumerations Implementation Specifics.
