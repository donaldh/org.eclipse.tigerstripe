Editing Notification Artifacts

After you successfully create the notification artifact, the corresponding icon appears in the Tigerstripe Explorer, and the Overview tab displays. You can further edit your newly created notification artifact in the Overview tab.

Editing a Notification Artifact

Note: Make sure the Tigerstripe Perspective is active.

To edit a notification artifact:

  1. Double-click the notification artifact in the Tigerstripe Explorer to open the Tigerstripe Notification Artifact editor.
  2. Edit the Notification Artifact as necessary and click Save to save your changes. Alternatively, you can select File and click Save or press CTRL+S to save your changes.

You can set the following parameters in the Tigerstripe Notification Artifact editor:

  • General Information: Common to all Artifacts, the general information section includes high level information such as version, and description.
  • Attributes: This is where you can define artifact attributes.
  • Constants: This is where you can define artifact constants.

Notification artifacts also include a set of options to customize the generated code according to OSS/J design guidelines. For more information, refer to Notification Implementation Specifics.
