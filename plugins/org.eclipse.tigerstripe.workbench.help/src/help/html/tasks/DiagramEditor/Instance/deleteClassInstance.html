Deleting a Class Instance

Tigerstripe Workbench enables you to delete instances of your diagram. When you delete an instance from the Class Instance Diagram you will not delete it from your Service Contract.

Note: If you delete something by accident from your Class Instance Diagram, you do have the ability to undo your changes up to the last Save performed. To undo any changes, right-click in the diagram, click Edit and select Undo. Alternatively, you can use the following shortcut keys:

  • Undo: CTRL+Z
  • Redo: CTRL+Y

To delete a class instance:

  1. Open the Class Instance Diagram by double-clicking on the .wod file in the Tigerstripe Explorer.
  2. Locate the instance that you want to delete in the Instance Diagram.
  3. Select the instance and press DELETE.
    -OR-
    Select the instance, click Edit and select Delete.
