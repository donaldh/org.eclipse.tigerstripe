Import/Export Tasks

Tigerstripe Workbench supports a number of different types of Import/Export options.

  • Importing and Exporting between Tigerstripe Projects
  • Exporting to XML from Tigerstripe Projects
  • Importing from XML to Tigerstripe Projects
  • Exporting to UML2 from Tigerstripe Projects
  • Importing from UML2 to Tigerstripe Projects
  • Importing from UML (1.4) to Tigerstripe Projects
  • Batch export of all project diagrams to .gif files
