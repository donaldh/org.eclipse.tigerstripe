Importing UML2 Profiles

The UML2 Profile Import tool will import UML Stereotype definitions and Primitive Type definitions from UML2 files into a Tigerstripe Profile. This is important before importing UML2 files, as Tigerstripe can only use Stereotype definitions and Primitive Types that exist in the Tigerstripe Profile at the time that you run the UML2 Import.

Running the Import

  1. From the Tigerstripe Perspective, right-click on the Tigerstripe Project into which you want to import Artifacts.
  2. Select Import.
  3. Navigate to the Tigerstripe/Import UML2 Profile and click Next. The Import UML2 Profile wizard opens.
  4. Select the Tigerstripe Profile to be imported (this will be a .wbp file in your workspace).
  5. Select the directory containing the ModelType and Profile files to be imported and click Finish.

The Tigerstripe Profile is updated with new Annotations and Primitive types.

Note: Remember that this profile must be deployed prior to an import that relies on these types of definitions and annotations.
