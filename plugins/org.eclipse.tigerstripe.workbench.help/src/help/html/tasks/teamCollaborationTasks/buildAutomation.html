Automated Build Integration ( Tigerstripe CLI )

Upon installation of Tigerstripe workbench, a new command is added to the list of available commands from the DOS command prompt. This new command allows to run the Tigerstripe engine on an existing Tigerstripe project and generate the corresponding Service Contract Integration profiles as configured in the project. This command allows to integrate the generation of the integration profiles from an automated script, eg.

From the command prompt

tigerstripe

will run the Tigerstripe engine.
