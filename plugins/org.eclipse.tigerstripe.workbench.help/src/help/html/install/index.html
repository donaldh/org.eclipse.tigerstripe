Tigerstripe Workbench Installation

This section provides insight on the installation and un-installation procedures for Tigerstripe Workbench. The minimal requirements for Tigerstripe Workbench 0.4 are:

  • JavaSE 5 or 6, which can be downloaded from http://java.sun.com.
  • Eclipse 3.4.1, which can be download from http://www.eclipse.org/downloads/. Any of the distributions that include JDT will work. For a minimal installation, we would recommend the 'Eclipse IDE for Java Developers'.
  • Windows XP, Mac or Linux.

We recomend that initial installation of Tigerstrip is done using manual installation. This the done as follows:

  • Go to the Tigerstripe downloads page.
  • Download the latest release of the Tigerstripe Workbench. Choose the 'bundle-tigerstripe-workbench...zip' file.
  • Unzip the file into the Eclipse 'dropins' folder. See The Eclipse Provisioning Wiki for more information.

Once, initial installation is completed, the same process can be used for updates or the Eclipse Update tool can be used. Please refer to the Eclipse User Guide for full/more details
