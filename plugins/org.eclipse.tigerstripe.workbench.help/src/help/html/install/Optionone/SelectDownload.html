Selecting the features to download

To let Eclipse select the correct dependencies for Tigerstripe Workbench, expand all top level nodes in the view, and expand the Tigerstripe Workbench site completely (a total of three levels). Complete the following procedure to select the features to download.

To download features:

  1. Select Tigerstripe Workbench and highlight the Workbench feature. Eclipse will complain about missing dependencies.
  2. Click Select Required to have Eclipse determine the dependencies for you, as shown in the following figure.

    3. Note: It is important that the top-level features in the list of features to install are expanded (see the example, below) when you click Select Required, otherwise the required features will not be selected.

  3. Click Next to start downloading all plug-ins.
  4. Select Install All, when prompted, to accept all license terms.
