User Defined Generation

In addition to the "built-in" plugins described here, a run a of Tigerstripe on a model can be configured to produce a set outputs based on their own or their company's specific standards. This can allow for company specific interface format, but can also allow for documentation, implementation artifacts and other information to be created from the same model, thus ensuring consistency.

User Defined Plugins

User defined plugins comprise a set of rules that define how model artifact and attributes are "rendered" in the output, what files are created etc, and a set of "Properties" that can be used to control the behaviour of the plugin. See the section Plugin Design Tasks for more information on how to create a plugin.
